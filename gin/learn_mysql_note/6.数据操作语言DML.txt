DML是对表中的数据进行增、删、改的操作

-------------------------增加数据--------------------------------
insert into 表名(列名1，列名2，列名3...) values(值1，值2，值3...),(值1，值2，值3); 
注意：日期和字符串要加单引号

向表中插入数据：
insert into student(name,age,sex) values('zhangsan',18,'nan');

插入多条数据：
insert into student(name,age,sex) values('zhangsan',18,'nan'),('lisi',20,'nan')

-------------------------------------------------------------------

--------------------------修改数据------------------------------------
update 表名 set 列名1=列值1,列名2=列值2，...

修改所有数据：
update student set name='lisi',age=20,sex='nv';

修改指定数据：加where条件
update student set id=2 where name='zhangsan';
-------------------------------------------------------------------


-------------------------删除数据---------------------------------

删除一条数据：
delete from student where name='wangwu';

清空表：不加条件
delete from student;

删除表后再创建新的空表，删除的数据不能找回：
truncate table student
